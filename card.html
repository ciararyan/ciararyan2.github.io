Design Process and Practice One4All Gift Card

Project Description

This assignment was also done using Adobe Illustrator. We had to design a gift card for a competition with One4All.My design is of an elephant because when I researched happiness on pinterest I seen a picture of an elephant and it just looked so happy and smiley. I thought this would look cute and interesting. My target audience for this gift card would be people buying a birthday present for any age as I was trying to make a birthday gift card with the quote “Age is Irrelephant” as this is also a pun.




Project Details

  • Happy and cute
